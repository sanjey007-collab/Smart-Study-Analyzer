# Smart Study Analyzer

A web application to track your study progress across multiple subjects.

## Live Demo
👉 **[Click here to use the website](https://smart-study-analyzer-x54q.onrender.com)**

## Features
- Subject Progress Tracking
- Performance Analytics
- Study Time Monitoring
- Goal Achievement Reports

## Tech Stack
- JSP / Java
- MySQL (Aiven Cloud)
- Tomcat
- Render (Hosting)

## How to Use
1. Register a new account
2. Login with your credentials
3. Add subjects and topics
4. View progress reports and insights
